Tabela de Operadores Aritméticos

Operador Descrição Exemplo
+ Soma de dois Valores 3 + 2 = 5
- Subtrai dois Valores 3 - 2 = 1
* Multiplica dois Valores 3 * 2 = 6
/ Divide dois valores 3 / 2 = 1.5
// Divide dois valores, porém arrendonda para baixo 3 / 2 = 1
% Resto da Divisão 3 % 2 = 1
** Exponenciação 3 ** 2 = 9

Tipos de Variáveis

  • Inteiro - Números inteiros - (int)
  • Ponto Flutuante - Números decimais - (float)
  • String - Qualquer tipo de caracter entre aspas - (str)
  • Boolean - Pode variar entre True or False - (bool)
  • List - É um conjundo de elementos que podem variar entre os tipos que estão listados a cima - (list)
  • Tuple - É a mesma coisa que a lista, porém os valores não podem ser alterados após sua definição - (list)
  • Dictionary - é um grupo de variaveis dentro de uma variavel pai - (dic)

Operador Descrição Exemplo
and "e" lógico Só se retorna "true" quando todas as condições são verdadeiras

not "não" lógico Inverte o valor da condição

Tabela de Operadores Relacionais

Operador Descrição Exemplo
== (bollean) Verifica se dois Valores são iguais 3 == 2 resulta em "False"
!= (bollean) Verifica se dois valores são diferentes 3 != 2 resulta em "True"
> (bollean) Verifica se x é maior que y 3 > 2 resulta em "True"
< (bollean) Verifica se x é menor que y 3 < 2 resulta em "False"
>= (bollean) Verifica se x é maior ou igual a y 3 =< 2 resulta em "False"
<= (bollean) Verifica se x é menor ou igual a y 3 <= 2 resulta em "False"
