Formatação de String.

String

    Uma sequencia de caracteres entre aspas simples ou aspas duplas configura uma string.

    Em Python, Strings são representadas como listas imutáveis de caracteres.

    Podemos representar uma String como uma sequência de caracteres entre aspas simples (') ou aspas duplas (").

    Exemplo:

                          
  x = 'Eu tenho 32'
  y = "anos de Idade"
  print(x + " " + y)

    Retorna:

                          
  Eu tenho 32 anos de Idade

    OBS: Quando concatenamos duas strings, caso queira espaço entre os dois valores das variaveis é nescessario atribuir uma string com espaço.

    Caso queira eliminar este processo, pois não nescessidade de concatenar, use virgulas entre as variaveis

Caracteres Especiais

dentro de uma string ainda sim podemos executar alguns comandos especias, dentre eles:

  • \ - Escapa um caracter de ser interpretado pela linguagem.
  • \n - Quebra de linha
  • \t - Tab equivalente a 8 espaços

    • Exemplo:

                          
  escapa = 'Dimitri\'s Burguer.'
  quebra = "Dimitri \né divertido."
  tab = "Dimitri é\tbonito."
  
  print(escapa)
  print(quebra)
  print(tab)

    Retorna:

                          
  Dimitri's Burguer.

  Dimitri 
  é divertido.

  Dimitri é       bonito.

String Especial

    Em strnigs também é possivel passar uma variavel sem o uso de virgulas ou concatenização de valores.

    Isso só é possivel declarando a letra "f" no fora da string, assim a string em questão ja esta definida como uma string especial.

    Exemplo:

                          
  idade = 10
  print(f'Eu tenho {idade} anos de idade')

    Retorna:

                          
  Eu tenho 10 anos de idade

Formatação de strings

    Uma das formas de formatar uma String é utilizando a função format.

    A função format recebe como parâmetros um valor e uma string com a formatação desejada.

    Para cada tipo de dao é nescessario um tipo de parametro diferente, entre eles estão:

  • d - Para Decimais
  • +d - Decimais com sinal
  • f - Para numeros reais
  • f+ - Para numeros reais com sinal
  • +.10f - Para numeros reais cmo sinais e prescisão

    • Exemplo:

                        
  print(format(3.14159265359, "+.5f"))

    Retorna:

                        
  +3.14159
