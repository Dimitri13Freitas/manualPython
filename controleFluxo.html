Controle de fluxo de execução.

Condicionais

Permitem que voçê fassa com que blocos de código sejam ou não executados em relaçao a condição colocada.

  • IF - Verifica se uma condição é verdadeira ou falsa
  • ELSE - Caso a primeira condição no IF seja falsa o bloco de código setado no else sera executado
  • ELIF - Em nescessidade de depois de um IF colocar mais uma condição se é usado o ELIF.

Laços de repetição

Executam um bloco de código quantas vezes for presciso, em efeito de uma condição ou quantidade setada

  • WHILE - Exige uma condição e enquanto essa condição for true o bloco de código setado é executado
  • FOR - é usado com objetos iteráveis tais como listas e intervalos
  • OBS: listas, loops e objetos semprem tem a contagem de itens ou loops começãda em zero
